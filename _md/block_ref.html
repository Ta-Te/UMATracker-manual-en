2.3.1. Color Filter Block¶

ColorFilter¶
Block Description
右のカラーパレットで色を選択し左の類似度（数値）を調整することで色を選択する．

注釈

ColorFilterブロックを選択後，『入力画面』上の抽出したい色の部分をクリックすることで色を選択することもできる．
